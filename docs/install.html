2010 » Installation Instructions

  1. Open license.txt and familiarize yourself with the terms under which you are permitted to use the software (GPLv3). Please pay particular attention to the part which attempts to shield me from liability if you screw up while using these steps.
  2. Download the budget CSV files from the Office of Management and Budget website or Data.gov. Currently available here: http://www.whitehouse.gov/omb/budget/Supplemental/
  3. Open docs/install.sql and change <REPLACE WITH PATH TO FILE> to the path to budgetauth.csv, outlays.csv, and receipts.csv.

    If you are on a shared server you might either need to do steps 3-4 on a local machine first and then export and import the resulting SQL, or use a different method to import the CSV file.
  4. Run docs/install.sql in MySQL. I used version 5, but 4 should work without much modification.
  5. Unpack the source files into a directory on a PHP capable webserver. PHP needs to be version 5.
  6. Open config.ini and change the MySQL settings to match your configuration.
  7. Hopefully the site should now work. If it doesn't, try commenting in the line "AddType x-mapp-php5 .php" in .htaccess by putting a pound sign (#) in front of it.
